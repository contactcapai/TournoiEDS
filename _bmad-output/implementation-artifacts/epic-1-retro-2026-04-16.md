## Défis

- **Pas de moyen simple de vérifier la BDD en local** — le Project Lead n'a pas pu vérifier si l'inscription était réellement persistée en base. Manque d'outillage dev (Prisma Studio non configuré).
- **Tests manuels Story 1.3 partiellement faits** — le flux frontend fonctionne (confirmation affichée) mais la persistance BDD n'a pas été vérifiée visuellement. Tasks 7.2 et 7.3 non cochées.
- **Responsive mobile signalé 2 fois en review** — nav hamburger manquant (Story 1.2, deferred) et H1 qui déborde (Story 1.4, patché).
- **Prisma 7 surprises** — import path (`../generated/prisma/client` pas `../generated/prisma`) et besoin de `@prisma/adapter-pg`. Mitigé par la documentation inter-stories.

## Enseignements clés

1. **La chaîne PRD → Architecture → Code fonctionne** — le rendu est fidèle aux attentes du Project Lead sans friction sur les technos.
